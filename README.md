# Logica_consultas_SQL
DataProject-LógicaConsultas SQL
Proyecto en el que se desarrollan 64 consultas de SQL, realizadas a través de DBeaver, utilizando la base de datos proporcionada por el equipo de thePower.
Lo primero fue cargar la BBDD, seguido de establecerla por defecto. 
Luego, crear el esquema Entidad Relación y actualizar la BBDD para que éste fuera visible.
A partir de ahí, fue comenzar a desarrollar una a una las consultas del enunciado.
